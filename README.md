# RecogniseChord — Ear Training for Chords & Intervals / Тренажер акордів і інтервалів

Коротко / Short
Тренувальник на вгадування акордів і інтервалів на слух. Веб‑додаток на Razor Pages (.NET 8) для практики розпізнавання акордів і інтервалів з браузерним відтворенням.

Features / Можливості
- Генерація акордів з конфігурованою кількістю звуків (2–5).
- Відтворення акордів у браузері (синтезоване аудіо / WAV).
- Вибір типу акорду та якості; перевірка відповідей із миттєвим фідбеком.
- Лічильник спроб; утиліти для роботи з аудіо та MIDI (папка `Utilities/`).

Requirements / Вимоги
- .NET 8 SDK
- Visual Studio 2022 (рекомендовано) або будь‑який текстовий редактор + CLI
- Браузер із підтримкою HTML5 audio

Getting Started / Швидкий старт
1. Клонуйте репозиторій:
git clone https://github.com/PianoMaker/RecogniseChord.git
2. В каталозі проекту запустіть:
dotnet restore dotnet run --project RecogniseChord

Або відкрийте рішення у Visual Studio 2022 і оберіть __Debug > Start Debugging__ або __Debug > Start Without Debugging__.

Usage / Як користуватися
- Відкрийте сторінку `/` — натисніть "Відтворити" для програвання згенерованого акорду.
- Оберіть кількість звуків, тип і якість акорду, потім натисніть "Перевірити".
- Результат показується з підказкою правильної відповіді; лічильник відслідковує спроби.

Configuration / Налаштування
- Параметри генерації та списки типів/якостей знаходяться в `RecogniseChord/Utilities/` (класи: `Chord`, `ChordT`, `Collections` тощо).
- Клієнтські змінні для відтворення: `window.generatedChord` і `window.generatedFile`.

Project Structure / Структура проекту
- `Pages/` — Razor Pages (UI). Головна сторінка — `Pages/Index.cshtml`.
- `Utilities/` — логіка генерації акордів, синтезу звуку, MIDI/втулки.
- `wwwroot/` — статичні файли (JS, CSS, аудіо).
- `Program.cs` — конфігурація додатку.

Troubleshooting / Відлагодження
- Якщо аудіо не відтворюється — перевірте консоль браузера; переконайтесь, що `window.generatedFile` або `window.generatedChord` не порожні.
- Помилки під час збірки — переконайтесь, що встановлено .NET 8 SDK.
